## Overview

The Agents tab browses custom AI personas, commands, and skills defined in your Claude Code setup.

## Features

- **Agents browser** showing all custom personas
- **Commands and Skills** listing with descriptions
- **Invocation stats** showing how often each agent/command is used
- **Detail view** with full agent configuration

## Agent Location

Agents are defined in `~/.claude/agents/` as markdown files with YAML frontmatter.
