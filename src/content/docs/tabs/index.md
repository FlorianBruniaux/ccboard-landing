## 9 Interactive Tabs

ccboard organizes Claude Code management into 9 focused tabs. Press `1-9` to jump directly to any tab.

| Tab | Name | Purpose |
|-----|------|---------|
| 1 | Dashboard | Key metrics, model usage, 7-day activity |
| 2 | Sessions | Browse sessions, live process detection |
| 3 | Config | Cascading config viewer (global/project/local) |
| 4 | Hooks | Event-based hook management with syntax highlighting |
| 5 | Agents | Custom AI personas and commands browser |
| 6 | Costs | Token analytics with 4 sub-views |
| 7 | History | Full-text search with temporal patterns |
| 8 | MCP | Server management with status detection |
| 9 | Analytics | Budget tracking, trends, insights, forecasting |

## Navigation

- `1-9`: Jump to specific tab
- `Tab`: Next tab
- `Shift+Tab`: Previous tab
- `j/k`: Move up/down in lists (vim-style)
- `/`: Search within current tab
- `r`: Refresh data
